# Simple rest server
네이버 검색 API를 실습하기 위한 간단한 REST SERVER 입니다.

## 레포지토리 클론
```shell
git clone https://github.com/Aqudi/Simple-rest-server.git
cd Simple-rest-server
```

## 패키지 설치
```shell
# 윈도우
pip install -r requirements.txt

# 우분투, Mac
pip3 install -r requirements.txt
```

## 서버 실행
``` shell
uvicorn main:app --reload  
```
